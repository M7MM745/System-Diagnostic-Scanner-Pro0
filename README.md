# 🔍 System Diagnostic Scanner - نظام ماسح التشخيص

تطبيق شامل لفحص جهازك من الألف إلى الياء يدعم اللغة العربية والإنجليزية

## 📋 المميزات / Features

✅ **Complete System Scan** - فحص شامل للنظام
✅ **Real-time Data** - بيانات فعلية من النظام
✅ **Arabic & English Support** - دعم اللغة العربية والإنجليزية
✅ **Hardware Info** - معلومات الأجهزة الكاملة
✅ **System Health Check** - فحص صحة النظام
✅ **Network Analysis** - تحليل الشبكة
✅ **Disk Space Monitoring** - مراقبة مساحة التخزين
✅ **Beautiful Charts** - رسوم بيانية تفاعلية
✅ **Export Reports** - تصدير التقارير
✅ **Mobile Responsive** - يعمل على الجوال والتابلت

## 🚀 كيفية الاستخدام / How to Use

### الطريقة 1: مع خادم Node.js (الأفضل - للبيانات الحقيقية)

#### الخطوة 1: تثبيت Node.js
إذا لم يكن لديك Node.js، قم بتحميله من [nodejs.org](https://nodejs.org)

#### الخطوة 2: تشغيل الخادم
افتح PowerShell أو Command Prompt في مجلد التطبيق:

```bash
cd "c:\Users\mhamad\Desktop\مجلد جديد (2)"
node server.js
```

سترى رسالة:
```
✅ System Diagnostic Server running on http://localhost:3000
```

#### الخطوة 3: افتح المتصفح
اذهب إلى: `http://localhost:3000`

**النتيجة:** 
- ✅ بيانات حقيقية من نظامك
- ✅ معلومات دقيقة عن المعالج والذاكرة والقرص
- ✅ معلومات الشبكة الفعلية
- ✅ جميع البيانات محدثة في الوقت الفعلي

---

### الطريقة 2: بدون خادم (بيانات محدودة)

يمكنك فتح `تست.html` مباشرة في المتصفح:

```
file:///c:/Users/mhamad/Desktop/مجلد جديد (2)/تست.html
```

أو استخدم Python Server:

```bash
cd "c:\Users\mhamad\Desktop\مجلد جديد (2)"
python -m http.server 8000
```

ثم افتح: `http://localhost:8000/تست.html`

**ملاحظة:** ستحصل على بيانات محدودة فقط

---

## 📱 على الجوال / On Mobile

### مع WiFi نفس الشبكة:

1. تأكد أن الجوال والكمبيوتر على نفس WiFi
2. ابحث عن IP address الكمبيوتر:
   ```bash
   ipconfig
   ```
3. على الجوال، ادخل في المتصفح:
   ```
   http://[IP_ADDRESS]:3000
   ```

مثال:
```
http://192.168.1.100:3000
```
